Dashboard inteligente para monitoreo **en tiempo real** de consumo eléctrico con alertas automáticas. **Proyecto de grado - Ingeniería de Sistemas.**

**[Live Demo →](https://andres-rs-1.github.io/proyectoGrado-backup/)**

## ✨ **Features**

- 📊 **Dashboard** consumo Watts/Volts/Corriente (live + histórico)
- 🚨 **Alertas Inteligentes** (alto/muy alto/crítico)
- 📈 **Estadísticas** 30 días + gráficos interactivos
- 💡 **Recomendaciones** ahorro energético personalizadas
- 🌍 **Impacto Ambiental** CO2 evitado + huella carbono
- 📱 **Responsive** Mobile/Desktop/Tablet
- 🌙 **Dark Mode** automático
- 🔊 **Sonidos + Notificaciones** push/toast
- ⚡ **Simulación IoT** datos realistas en tiempo real

## 🛠️ **Tech Stack**
Frontend: React 18 + TypeScript + Vite + shadcn/ui
Estilos: Tailwind CSS v3 + Lucide Icons
Gráficos: Recharts + Framer Motion
Notificaciones: Sonner
Docker: Multi-stage build
Deploy: GitHub Pages + Docker Hub


## 🎯 **Funcionalidades Clave**

| Pantalla         | Features                                      |
|------------------|-----------------------------------------------|
| **Login**        | Autenticación hogar/local                     |
| **Config**       | Umbral alertas + simulación IoT configurable  |
| **Dashboard**    | Métricas live + navegación rápida             |
| **Alertas**      | Filtros + Switch "solo no leídas"             |
| **Estadísticas** | Gráficos 24h/7d/30d + export PDF/CSV         |
